1. 개요

Q-Track 백엔드 API 문서입니다.

2. 인증

모든 API는 JWT 토큰 기반 인증을 사용합니다. 요청 헤더에 Authorization: Bearer {token} 형식으로 토큰을 포함해야 합니다.

2.1. 로그인

로그인 API를 통해 JWT 토큰을 발급받을 수 있습니다.

Content-Length: 62
Host: localhost:8080

{
  "username" : "testuser",
  "password" : "password123"

Pragma: no-cache

  "status" : 200,
  "msg" : "성공",
  "body" : {
    "token" : "eyJhbGciOiJIUzUxMiJ9.eyJzdWIiOiIxMTEiLCJyb2xlIjoiVVNFUiIsImlhdCI6MTc2ODY2NTEzMSwiZXhwIjoxNzY5MDk3MTMxfQ.tTFI2DlRkQFYLUyvys3USO_p__fYZTf3xV0E-CfvuHCJ1vD3FFfduPCDWX3zNJZCxkqr2s4YrCzKYEXJVz8Bzw",
